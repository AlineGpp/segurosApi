## O que o sistema faz

#### Sistema para gerenciamento de apólices e clientes:

Funcionalidades:

* cadastrar clientes
* cadastrar seguros
* criar apólices
* listar contratos
* atualizar dados
* cancelar apólices
* integração entre entidades

## Stack

#### Backend:

* C#
* ASP.NET Core Web API
* Entity Framework
* SQL Server
* Swagger
* JWT
* Repository Pattern
